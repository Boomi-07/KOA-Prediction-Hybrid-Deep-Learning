# KOA Prediction Using Hybrid Deep Learning Algorithm

## Overview

This project focuses on predicting Knee Osteoarthritis (KOA) using a hybrid deep learning approach. The model is designed to classify knee X-ray images and assist in the early detection of osteoarthritis.

## Project Status

This repository currently contains the initial project report. The source code, trained model, dataset information, and additional documentation will be uploaded in future updates.

## Features

* Knee Osteoarthritis prediction
* Deep learning-based image classification
* Model training and evaluation
* Performance analysis using metrics such as accuracy, confusion matrix, ROC curve, and precision-recall curve

## Results

The current implementation achieved an accuracy of **92.78%** on the validation dataset.
